================================================================================
TROUBLESHOOTING
================================================================================

Problem: "ModuleNotFoundError"
Solution: pip install -r requirements.txt

Problem: "RAG system not loaded" warning in chatbot
Solution: Run setup_rag.py first

Problem: "db folder not found"
Solution: Run setup_rag.py to create it

Problem: Chatbot not responding to detailed questions
Solution: Check that setup_rag.py completed successfully


================================================================================
OPTIONAL: IMPROVE AI RESPONSES
================================================================================

For better AI responses, add HuggingFace API token:

1. Go to: https://huggingface.co/settings/tokens
2. Create free account
3. Create new token
4. Create .env file in project folder:
   
   HUGGINGFACE_API_TOKEN=your_token_here

5. Restart chatbot

Note: Chatbot works without token, but responses may be limited.
