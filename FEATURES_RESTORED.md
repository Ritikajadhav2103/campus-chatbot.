## 🆕 New Feature: Dark Mode Toggle

### What's New
- 🌙 Dark mode with deep blue-black theme
- ☀️ Light mode with clean white/gray theme
- 🔄 One-click toggle button in header
- ✨ Smooth color transitions (0.3s)
- 💾 Theme persists during session

### Dark Mode Colors
```
Background: #1a1a2e (Deep blue-black)
Cards: #16213e (Navy blue)
Text: #eaeaea (Light gray)
Secondary: #a0a0a0 (Medium gray)
Bot Messages: #2d3748 (Dark gray)
Borders: #3a3a52 (Subtle gray)
```

### Light Mode Colors
```
Background: #f8f9fa (Light gray)
Cards: #ffffff (White)
Text: #212529 (Dark gray)
Secondary: #6c757d (Medium gray)
Bot Messages: #f1f3f5 (Light gray)
Borders: #e9ecef (Light border)
```

### How to Use
1. Look for the button in top-right corner of header
2. Click 🌙 Dark to switch to dark mode
3. Click ☀️ Light to switch back to light mode
4. Theme stays active during your session
