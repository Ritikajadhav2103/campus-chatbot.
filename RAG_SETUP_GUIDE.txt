WHAT IS RAG?
------------
RAG (Retrieval Augmented Generation) allows the chatbot to answer detailed 
questions by reading and understanding the handbook content using AI.

WHAT CAN YOU ASK NOW?
----------------------
With RAG enabled, you can ask detailed questions like:

✅ "What are the admission requirements for engineering?"
✅ "Tell me about fees at Walchand Institute"
✅ "What facilities are available at WIT?"
✅ "Placement information for engineering colleges"
✅ "Hostel details and accommodation"
✅ "What is the eligibility criteria?"
✅ "How to apply for admission?"
✅ "Tell me about scholarships"
✅ "What are the course durations?"
✅ "Faculty information"

SETUP STEPS (FIRST TIME ONLY)
------------------------------

STEP 1: Set HuggingFace API Token (Optional but Recommended)
-------------------------------------------------------------
For better AI responses, get a free API token:

1. Go to: https://huggingface.co/settings/tokens
2. Create account (free)
3. Create new token
4. Copy the token

5. Create .env file in project folder:
   HUGGINGFACE_API_TOKEN=your_token_here

Note: Without token, chatbot will still work but with limited AI capabilities.


STEP 2: Run Setup Script
-------------------------
Open command prompt in project folder and run:

    python setup_rag.py

This will:
- Load the handbook
- Split it into chunks
- Create embeddings
- Build vector store
- Test the system

Expected output:
✓ Found handbook
✓ Loaded documents
✓ Created chunks
✓ Embeddings model loaded
✓ Vector store created
✓ Test successful
✅ RAG SETUP COMPLETE!


STEP 3: Run the Chatbot
------------------------
Double-click: RUN_RAG_CHATBOT.bat

Or run manually:
    python -m streamlit run app_solapur_rag.py

Browser will open at: http://localhost:8501

TROUBLESHOOTING
---------------

Problem: "RAG system not loaded" warning
Solution: Run setup_rag.py first

Problem: Slow responses
Solution: Add HuggingFace API token to .env file

Problem: "db folder not found"
Solution: Run setup_rag.py to create vector store

Problem: Import errors
Solution: Install requirements:
    pip install -r requirements.txt
