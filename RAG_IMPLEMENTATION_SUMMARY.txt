HOW IT WORKS
================================================================================

INTELLIGENT QUESTION ROUTING:

1. User asks question
   ↓
2. Chatbot analyzes question type
   ↓
3. Routes to appropriate system:
   
   COLLEGE DATABASE (Fast):
   - "Tell me about WIT" → Shows college card
   - "Engineering colleges" → Shows list
   
   RAG SYSTEM (Intelligent):
   - "What are admission requirements?" → AI reads handbook
   - "Tell me about fees" → AI extracts information
   - "Placement details" → AI provides answer


TECHNICAL ARCHITECTURE
================================================================================

Components:
-----------
1. College Database (JSON)
   - 35+ colleges
   - Basic information
   - Fast lookup

2. Vector Store (Chroma)
   - Handbook content in chunks
   - Semantic search
   - Fast retrieval

3. Embeddings (HuggingFace)
   - sentence-transformers/all-MiniLM-L6-v2
   - Converts text to vectors
   - Enables semantic search

4. LLM (HuggingFace)
   - mistralai/Mistral-7B-Instruct-v0.2
   - Generates answers
   - Understands context

5. LangChain
   - Orchestrates RAG pipeline
   - Manages retrieval and generation
   - Handles prompts


DATA FLOW
================================================================================

Question: "What are the admission requirements?"

1. Question received
   ↓
2. Detected as detailed question
   ↓
3. Vector store searches handbook
   ↓
4. Retrieves 3 most relevant chunks
   ↓
5. LLM reads chunks + question
   ↓
6. Generates natural language answer
   ↓
7. Answer displayed to user


SETUP PROCESS
================================================================================

STEP 1: Install Dependencies
-----------------------------
All required libraries in requirements.txt:
- langchain
- langchain-community
- langchain-huggingface
- sentence-transformers
- chromadb
- streamlit
- etc.

STEP 2: Run Setup Script
-------------------------
python setup_rag.py

This:
1. Loads data/solapur_colleges_handbook.txt
2. Splits into 1000-char chunks with 200-char overlap
3. Creates embeddings for each chunk
4. Stores in Chroma vector database (db/ folder)
5. Tests retrieval

STEP 3: Run Chatbot
--------------------
python -m streamlit run app_solapur_rag.py

Chatbot loads:
1. College database (cached)
2. Vector store (cached)
3. LLM connection
4. UI components
