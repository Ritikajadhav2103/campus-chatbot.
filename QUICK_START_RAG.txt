                SOLAPUR COLLEGES CHATBOT - QUICK START
================================================================================

🚀 GET STARTED IN 3 STEPS!

STEP 1: Setup RAG System (First Time Only)
-------------------------------------------
Open command prompt and run:

    python setup_rag.py

Wait for completion (2-3 minutes)
You should see: ✅ RAG SETUP COMPLETE!


STEP 2: Run the Chatbot
------------------------
Double-click: RUN_RAG_CHATBOT.bat

Or run manually:
    python -m streamlit run app_solapur_rag.py


STEP 3: Start Asking Questions!
--------------------------------
Browser opens at: http://localhost:8501

Try these questions:
✅ "Tell me about Walchand Institute of Technology"
✅ "What are the admission requirements?"
✅ "Engineering colleges in Solapur"
✅ "Tell me about fees"
✅ "Placement information"


THAT'S IT! 🎉
