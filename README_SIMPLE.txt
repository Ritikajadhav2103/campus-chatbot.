🚀 EASIEST WAY TO RUN (Windows)
═══════════════════════════════════════════════════════════════
  1. Double-click: RUN_CHATBOT.bat
  2. Wait for browser to open
  3. Start chatting!

📝 MANUAL WAY TO RUN
═══════════════════════════════════════════════════════════════
  1. Open Command Prompt or PowerShell
  
  2. Go to project folder:
     cd C:\Users\ritik\OneDrive\Desktop\Campus_info_chatbot
  
  3. Activate virtual environment:
     venv\Scripts\activate
  
  4. Run the app:
     python -m streamlit run app_history.py
  
  5. Open browser:
     http://localhost:8501

🔧 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════
  Problem: Python not found
  Solution: Install Python 3.12 from python.org
  
  Problem: Packages not installed
  Solution: Run "pip install -r requirements.txt"
  
  Problem: Port already in use
  Solution: Close other Streamlit apps or use different port
